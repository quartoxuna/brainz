# Brainz - Brainfuck Interpreter
A simple Brainfuck Interpreter written in python

# Usage:
```
python brainz.py "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.+++."
CODE   : ++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>.+++.
( 15)                   ^
STACK  : [  9] [ 11] [ 10] [  3] [  1]
(  1)            ^                    
STDOUT : ''
STDIN  : ''
```
